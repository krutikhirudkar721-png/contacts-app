# contacts-app
# PROBLEM STATEMENT :- 
<br>
Contacts List<br>
Ready to tackle a functioning app instead of a game? Why not try building a contacts list, similar to the one available on your phone?<br>
This project represents a new challenge since you'll be asking for a variety of string inputs and then allowing the user to interact with them (update, search, delete, etc.). First, you'll create the parameters for a contacts list, and then you'll provide the ability to manipulate them. That's where this project grows more ambitious.<br>
Skills Tested<br>
You'll work to become comfortable with classes, lists, strings, and functions in Python to create a contacts list. The code will contain plenty of "if" and "else" statements, which can get tricky to track but are well worth the effort.<br>


# Features:
<br>
Add new contacts with name and phone number
<br>
View all saved contacts
<br>
Search contacts by name
<br>
Update existing contact numbers
<br>
Delete contacts
<br>
Persistent storage using JSON
<br>
Technologies: Python, OOP, JSON
